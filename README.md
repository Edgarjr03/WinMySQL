# WinMySQL / MySQL Desktop App

## Español
Aplicación de escritorio desarrollada en C# que se conecta a una base de datos MySQL
ejecutada en un contenedor Docker. Permite gestionar y visualizar datos mediante
una interfaz gráfica.

### Tecnologías
- C#
- Windows Forms
- Visual Studio 2022
- MySQL Workbench
- Docker

### Requisitos
- Tener Docker instalado y corriendo
- Tener MySQL Workbench instalado

### Cómo ejecutar
1. Clona el repositorio
2. Levanta el contenedor de MySQL con Docker
3. Abre el archivo `.sln` en Visual Studio
4. Configura la cadena de conexión con tu IP y puerto
5. Presiona `F5` para ejecutar

## English
Desktop application developed in C# that connects to a MySQL database
running in a Docker container. Allows managing and visualizing data
through a graphical interface.

### Technologies
- C#
- Windows Forms
- Visual Studio 2022
- MySQL Workbench
- Docker

### Requirements
- Have Docker installed and running
- Have MySQL Workbench installed

### How to run
1. Clone the repository
2. Start the MySQL container with Docker
3. Open the `.sln` file in Visual Studio
4. Configure the connection string with your IP and port
5. Press `F5` to run
